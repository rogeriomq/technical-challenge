# **Challenge - SPA**
> This SPA(Single Page Application) use React, Vite, Typescript

## Instalation
**Requiriments**
- NodeJS 16.x
- NPM 8.x or Yarn 1.22.x

### Using NPM
Install packages:
 ```
$ npm install
 ```
### Run Project

On dev run:
```bash
$ npm run dev
```

To generate the bundle/dist:
```bash
$ npm run dist
```

Have fun!
